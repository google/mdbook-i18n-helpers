## Installation

- ### [mdbook-i18n-helpers](./i18n-helpers/README.md):

  Run

  ```shell
  $ cargo install mdbook-i18n-helpers
  ```

  Please see [USAGE](./i18n-helpers/USAGE.md) for how to translate your `mdbook`
  project.
